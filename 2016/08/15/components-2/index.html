Components, Part 2

Nest Learning Thermostat, Gen 3

This is a smart thermostat that connects to your wireless network and allows for control, via the “cloud”, over all aspects of your home HVAC. Another notable competitor is EcoBee, which includes remote sensors and other great features, but the EcoBee requires a “C” wire in order to be powered, which was not an option for me in my current environment. The neat part about the Nest is that it trickle charges by pulling small amounts of power from the already existing systems, so a new wire run is not necessary.

It integrates really well with the Echo after installing the “Nest” skill. In the default state, if you ask it to turn the temperature or thermostat down one degree, it fails, so I worked around this by creating called “Thermostat”, “Temperature”, and “Nest” in order for the Echo to effectively respond to any commands I would throw at it.
