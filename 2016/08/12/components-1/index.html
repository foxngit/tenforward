Components, Part 1

These are the components that I already have. Just establishing a baseline here before I start adding on everything else!

Amazon Echo

I got a great deal on this unit ($109.99) at Home Depot through a single day promotion that was running almost everywhere but Amazon. It was interesting to learn that Amazon was not pricematching their own item during a promotional event. Even more interesting was to learn that Home Depot carries products like this.

As of today, there is still a promotional deal running where adding an Echo and Philips Hue starter kit to your cart and applying the code ‘ECHO40PHUE’ for an additional $40 off.

Anyways, this thing is kind of the “hub” of all voice activation features behind the home automation project. You can create “skills” to integrate other smart home devices with the Echo, and then use voice commands to control them all.

Link: Home Depot

Philips Hue

I bought this at the same time as the Echo, along with two extra bulbs. The hub requires a physical (ethernet) connection to your home network. I plugged mine into a switch. It is then easily configurable from any smartphone that is connected to the same network. Hue allows for super granular control over any lightbulbs associated with the system. Some people do not care for the Philips approach here because it only works with Philips bulbs, but I am starting from scratch. As of the time of this posting, the promotion mentioned above still works.

One of our favorite features is being able to dim the bulbs in lamps that did not previously support dimming. Mood lighting, anyone?

Link: Home Depot
